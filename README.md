# South Park : Coon and firends

Для запуска :

- .Net Framework 4.0
- XNA Game Studio 4.0

!!! Шрифт нужно установить! (пока сам не инсталится)

Доработано :

1 Новая панель управления(изящьная, дерзка, как South park резкая). :))
2 Переработаны иконки Эрика Картмана.
3 Изменена анимация и реакция на различное колличество здоровья.
4 Добавлен класс ManagerCheckPoints - управляет контрольными точками(по которым двигаются враги).
5 Переработаны классы CheckPointStart, CheckPointMove, CheckPointFinish.
6 Добавлен класс ManagerEnemies - управляет процессом создания врагов. 
7 Улучшена работа класса ManagerCollision -  обрабатывает коллизии.
8 Отрисовка игрового интерфейса(панелек, всяких индикаторов и т.д. и т.п.) вынесено в отдельный класс - ManagerDrawingInterface.
9 Теперь у башенек есть область видимости.
10 Если враг попадает в область видимости башни - последняя поворачивается в сторону врага.
11 Если враг атакует башню, то она поварачивается в его сторону(если не была повернута в ту сторону)
12 Добавлена "куча" звуков и фраз Эрика.
13 Ну и различные мелочи... ))
